## Features Implemented

- Server Actions with useOptimistic for instant feedback
- Debounced hybrid search (SSR + Client API)
- Race-condition-safe status toggling
- Soft delete with modal confirmation
- Real-time synchronized audit logs
- Authentication with user-based data isolation
- Loading and mutation UX states
- Empty state handling
- Modern dashboard UI

## Tech Stack

- Next.js (App Router)
- MongoDB
- Tailwind CSS v4
- NextAuth
- Zod Validation
